AGI Personal Memory

MITalpha

Accumulative verified memory substrate for language models.

Incremental, auditable, reversible knowledge accumulation. One fact at a time, verified before commit.

Quick Start

pip install -e .
agim teach "Paris is the capital of France"
agim ask "What is the capital of France?"
agim correct "No, Napoleon was born in 1769"
agim history
agim stats
agim shell

How It Works

User Input → IntentRouter → MemoryExtractor → MemoryCandidate
                                              ↓
                                         VERIFY (5 gates)
                                              ↓
                                    MemoryCompiler (tier)
                                              ↓
                                    ┌─ WAL recipe (permanent)
                                    ├─ Retrieval (volatile)
                                    └─ Reject / Refusal
                                              ↓
                                    COMMIT + Event Log

Key Properties

PropertyHow
IncrementalOne fact at a time, not batch fine-tuning
VerifiedEvery change tested before commit
ReversibleRollback any commit
AuditableJSONL event log of every action
Non-destructiveFrozen vocabulary = 0% non-target diff
