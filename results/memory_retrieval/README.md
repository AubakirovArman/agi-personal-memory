# Memory/Retrieval Results

These artifacts measure Path A behavior: storing and retrieving memories through
AGIM's memory layer, JSON stores, BM25/FAISS-like retrieval, and LoCoMo/SQuAD
recall experiments.

They do not measure whether model weights were edited. Do not compare these
numbers to EasyEdit, ROME, MEMIT, AlphaEdit, or other weight-editing methods.

Historical CounterFact/weight-editing summaries were moved to
`../local_protocol/`.
