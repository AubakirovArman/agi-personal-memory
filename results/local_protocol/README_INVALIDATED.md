# Invalidated For Official EasyEdit Claims

The artifacts in this folder are historical AGIM-local diagnostics.

They are preserved for audit and regression archaeology, but they must not be
cited as official EasyEdit results or leaderboard evidence.

Use this wording:

```text
Historical AGIM-local CounterFact diagnostic.
```

Do not use this wording:

```text
Official EasyEdit result.
Leaderboard score.
Solved sequential/lifelong editing.
```
